JavaScript ES6+: Những tính năng hiện đại bạn phải biết

29/12/2025

ES6 là gì?

ES6 (ECMAScript 2015) là bước nhảy vọt lớn nhất lịch sử JavaScript. Kể từ phiên bản này, JS trở nên mạnh mẽ, rõ ràng và phù hợp để xây dựng các ứng dụng lớn. Dưới đây là các tính năng “must-know”.

1. let và const

Tạm biệt var.

  • let: Khai báo biến có thể thay đổi giá trị. Phạm vi Block scope {}.
  • const: Khai báo hằng số. Không thể gán lại giá trị. Phạm vi Block scope.
if (true) {
    let a = 10;
    var b = 20;
}
// console.log(a); // Lỗi: a is not defined
console.log(b); // 20 (Do var có function/global scope)

2. Arrow Function

Cú pháp hàm mũi tên ngắn gọn hơn và xử lý từ khóa this khác biệt (lexical this).

// Cũ
const sum = function(a, b) {
    return a + b;
};

// Mới
const sum = (a, b) => a + b;

3. Template Literals

Dùng dấu backtick (`) để tạo chuỗi đa dòng và chèn biến siêu dễ.

const name = "Hào";
const msg = `Xin chào ${name},
Chào mừng bạn đến với Blog!`;

4. Destructuring (Phân rã)

Trích xuất dữ liệu từ Array hoặc Object cực nhanh.

// Object
const user = { name: "Hào", age: 22 };
const { name, age } = user;

// Array
const colors = ["Red", "Green"];
const [first, second] = colors;

5. Default Parameters

function welcome(name = "Khách") {
    console.log(`Hello ${name}`);
}
welcome(); // Hello Khách

6. Spread Operator (…)

Dùng để sao chép hoặc gộp mảng/object.

const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // [1, 2, 3, 4]

const userUpdate = { ...user, email: "new@email.com" };

7. Modules (Import/Export)

Chia nhỏ code thành nhiều file để dễ quản lý.

// lib.js
export const pi = 3.14;

// main.js
import { pi } from './lib.js';

Kết luận

Nếu bạn vẫn đang viết JS kiểu cũ (ES5), hãy cập nhật ngay lên ES6+. Đây là tiêu chuẩn của mọi framework hiện đại như React, Vue và Angular.
