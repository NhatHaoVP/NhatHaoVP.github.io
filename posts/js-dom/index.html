Làm chủ JavaScript DOM Manipulation

29/12/2025

DOM là gì?

DOM (Document Object Model) là cách trình duyệt biểu diễn trang HTML của bạn dưới dạng một cây (tree) các đối tượng. JavaScript thông qua DOM để thay đổi nội dung, cấu trúc và kiểu dáng của trang web một cách động.

1. Chọn phần tử (Selecting Elements)

Quên getElementById hay getElementsByClassName đi nếu bạn muốn code hiện đại. Hãy dùng bộ đôi quyền lực:

  • document.querySelector('selector'): Chọn phần tử đầu tiên khớp với CSS selector.
  • document.querySelectorAll('selector'): Chọn TẤT CẢ phần tử khớp (trả về NodeList).
const title = document.querySelector('#main-title');
const buttons = document.querySelectorAll('.btn-click');

2. Thay đổi nội dung và Style

const box = document.querySelector('.box');

// Đổi nội dung
box.textContent = "Nội dung mới"; // An toàn hơn innerHTML
// box.innerHTML = "<b>Đậm</b>"; // Cẩn thận XSS attack

// Đổi Style trực tiếp
box.style.backgroundColor = "red";
box.style.fontSize = "20px";

// Thêm/Xóa Class (Khuyên dùng thay vì sửa style trực tiếp)
box.classList.add('active');
box.classList.remove('hidden');
box.classList.toggle('dark-mode');

3. Lắng nghe sự kiện (Event Listeners)

Đừng bao giờ dùng onclick="..." trong HTML. Hãy tách biệt logic ra khỏi giao diện bằng addEventListener.

const btn = document.querySelector('#submit-btn');

btn.addEventListener('click', function(event) {
    console.log("Đã click nút!");
    console.log(event.target); // Phần tử bị click
});

4. Sự kiện nổi bọt (Event Bubbling) & Ủy quyền (Delegation)

Khi bạn click một thẻ con, sự kiện sẽ “nổi” lên các thẻ cha. Ta có thể tận dụng điều này để xử lý sự kiện cho nhiều phần tử con bằng cách chỉ gán 1 listener cho cha (Event Delegation).

// Thay vì gán click cho 100 thẻ <li>
const list = document.querySelector('ul');

list.addEventListener('click', (e) => {
    if (e.target.tagName === 'LI') {
        console.log("Bạn đã chọn mục:", e.target.textContent);
    }
});

Cách này giúp tiết kiệm bộ nhớ và code chạy nhanh hơn.

Kết luận

Thao tác DOM thuần thục giúp bạn hiểu sâu về cách trình duyệt hoạt động, kể cả khi sau này bạn chuyển sang dùng React hay Vue, kiến thức nền tảng này vẫn cực kỳ quý giá.
