Nhập môn Spring Boot: Tại sao nên dùng?

29/12/2025

Spring vs Spring Boot

Spring Framework là một framework khổng lồ và mạnh mẽ nhất thế giới Java. Tuy nhiên, việc cấu hình Spring (XML hoặc Java Config) rất phức tạp và tốn thời gian (“Configuration Hell”).

Spring Boot ra đời để giải quyết vấn đề đó. Nó được xây dựng trên nền tảng Spring nhưng đi kèm triết lý “Convention over Configuration” (Quy ước hơn cấu hình).

  • Auto-configuration: Tự động cấu hình các thư viện dựa trên file JAR có trong project.
  • Standalone: Tích hợp sẵn server (như Tomcat, Jetty) nên có thể chạy ứng dụng chỉ bằng lệnh java -jar.
  • Opinionated: Đưa ra các cấu hình mặc định hợp lý để bạn bắt đầu code business logic ngay lập tức.

Các khái niệm cốt lõi

1. Inversion of Control (IoC) và Dependency Injection (DI)

Đây là trái tim của Spring. Thay vì bạn tự tạo đối tượng (new Object()), Spring Container sẽ tạo và quản lý chúng (gọi là Beans). Khi cần dùng, Spring sẽ tự “tiêm” (inject) vào cho bạn.

@Service // Đánh dấu đây là một Bean
public class UserService { ... }

@RestController
public class UserController {
    @Autowired // Spring tự động tìm UserService và đưa vào đây
    private UserService userService;
}

2. Các Annotation phổ biến

  • @Component, @Service, @Repository: Đánh dấu class là Bean để Spring quản lý.
  • @Autowired: Tiêm phụ thuộc (DI).
  • @RestController: Đánh dấu class xử lý API request và trả về JSON.
  • @RequestMapping, @GetMapping, @PostMapping: Định tuyến URL đến hàm xử lý.

3. Application Properties

Mọi cấu hình (database, server port, logging…) đều nằm gọn trong file application.properties hoặc application.yml.

server.port=8080
spring.datasource.url=jdbc:mysql://localhost:3306/mydb
spring.datasource.username=root
spring.datasource.password=1234

Ví dụ: Hello World API

Chỉ với vài dòng code, bạn đã có một REST API hoàn chỉnh:

@RestController
public class HelloController {

    @GetMapping("/hello")
    public String sayHello() {
        return "Xin chào! Spring Boot rất tuyệt!";
    }
}

Truy cập http://localhost:8080/hello sẽ nhận được chuỗi text trên.

Kết luận

Spring Boot đã thay đổi cuộc chơi của Java Backend Developer, giúp việc phát triển Microservices trở nên dễ dàng và nhanh chóng hơn bao giờ hết. Nếu bạn làm Java Web, Spring Boot là kỹ năng bắt buộc.
