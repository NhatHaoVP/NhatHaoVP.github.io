Java Cơ Bản: Biến và Kiểu Dữ Liệu

29/12/2025

Giới thiệu về Java

Java là một ngôn ngữ lập trình hướng đối tượng, đa mục đích và mạnh mẽ, được sử dụng rộng rãi trong phát triển phần mềm doanh nghiệp, ứng dụng di động (Android) và hệ thống lớn. Một trong những khái niệm nền tảng quan trọng nhất khi bắt đầu học Java chính là biến và các kiểu dữ liệu.

Việc hiểu rõ cách Java lưu trữ và xử lý dữ liệu sẽ giúp bạn viết code hiệu quả, tối ưu bộ nhớ và tránh các lỗi logic không đáng có.

Biến (Variables)

Biến là tên gọi của một vùng nhớ dùng để lưu trữ dữ liệu. Trong Java, vì là ngôn ngữ định kiểu tĩnh (statically typed), bạn bắt buộc phải khai báo kiểu dữ liệu cho biến trước khi sử dụng.

Cú pháp khai báo

// KieuDuLieu tenBien = GiaTri;
int age = 20;
String name = "Hào";
double salary = 1000.50;
boolean isStudent = true;

Quy tắc đặt tên biến (Naming Convention)

Trong Java, việc đặt tên biến tuân theo quy tắc camelCase:

  • Ký tự đầu tiên viết thường.
  • Các từ tiếp theo viết hoa chữ cái đầu.
  • Tên biến nên có ý nghĩa rõ ràng (ví dụ: customerName thay vì n).

Phạm vi của biến (Variable Scope)

  1. Local Variables (Biến cục bộ): Khai báo trong phương thức, constructor hoặc block. Chỉ tồn tại trong phạm vi đó.
  2. Instance Variables (Biến toàn cục): Khai báo trong class nhưng ngoài phương thức. Thuộc về đối tượng (object).
  3. Static Variables (Biến tĩnh): Khai báo với từ khóa static. Thuộc về class, dùng chung cho tất cả đối tượng.

Kiểu dữ liệu (Data Types)

Java chia kiểu dữ liệu thành hai nhóm chính: Primitive (Nguyên thủy) và Reference (Tham chiếu).

1. Primitive Data Types (Kiểu nguyên thủy)

Java có 8 kiểu dữ liệu nguyên thủy, lưu trữ giá trị trực tiếp và tốn ít bộ nhớ.

Kiểu Kích thước Mô tả Ví dụ
byte 1 byte Số nguyên rất nhỏ (-128 đến 127) byte b = 100;
short 2 bytes Số nguyên nhỏ short s = 10000;
int 4 bytes Số nguyên thông dụng nhất int i = 100000;
long 8 bytes Số nguyên rất lớn (cần hậu tố L) long l = 100000L;
float 4 bytes Số thực (cần hậu tố f) float f = 5.75f;
double 8 bytes Số thực độ chính xác cao double d = 19.99;
boolean 1 bit Giá trị logic (true/false) boolean flag = true;
char 2 bytes Một ký tự Unicode char Grade = 'A';

2. Reference Data Types (Kiểu tham chiếu)

Kiểu tham chiếu được dùng để lưu trữ địa chỉ của đối tượng trong bộ nhớ Heap.

  • String: Chuỗi ký tự (là một object đặc biệt).
  • Arrays: Mảng chứa nhiều phần tử.
  • Classes & Interfaces: Các đối tượng do người dùng định nghĩa.

Ép kiểu (Type Casting)

Trong Java, việc chuyển đổi từ kiểu dữ liệu này sang kiểu dữ liệu khác gọi là Type Casting.

Implicit Casting (Nới rộng)

Chuyển từ kiểu nhỏ sang kiểu lớn hơn, diễn ra tự động. byte -> short -> char -> int -> long -> float -> double

int myInt = 9;
double myDouble = myInt; // Tự động chuyển thành 9.0

Explicit Casting (Thu hẹp)

Chuyển từ kiểu lớn sang kiểu nhỏ hơn, phải làm thủ công và có thể mất dữ liệu.

double myDouble = 9.78;
int myInt = (int) myDouble; // Thủ công, kết quả là 9 (mất phần thập phân)

Kết luận

Nắm vững các kiểu dữ liệu và biến là bước đệm vững chắc để bạn tiến sâu hơn vào lập trình Java. Hãy chú ý đến việc chọn kiểu dữ liệu phù hợp để tối ưu hóa hiệu năng chương trình của bạn!
